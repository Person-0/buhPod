# BuhPod PCB

- These are the PCB design files, made using KiCAD.<br>
- This is technically a fork of [Minimal design example](https://datasheets.raspberrypi.com/rp2350/Minimal-KiCAD.zip) provided by raspberrypi, in their [hardware design docs for the RP2350](https://pip-assets.raspberrypi.com/categories/1214-rp2350/documents/RP-008280-DS-1-hardware-design-with-rp2350.pdf).
- It is currently WIP.
